永久連結（Permalinks）

您可以在設定中調整網站的永久連結。

變數

除了下列變數外，您還可使用 Front-matter 中的所有屬性。

變數描述
:year文章的發表年份（4 位數）
:month文章的發表月份（2 位數）
:i_month文章的發表月份（去掉開頭的零）
:day文章的發表日期 (2 位數)
:i_day文章的發表日期（去掉開頭的零）
:title檔案名稱
:id文章 ID
:category分類。如果文章沒有分類，則是 default_category 設定。
